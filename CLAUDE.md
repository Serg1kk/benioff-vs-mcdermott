# Benioff vs McDermott: Death Match

A neon cyberpunk top-down arena shooter.

## Project Structure

- `index.html` - Single file game (HTML + CSS + JS)
- `docs/plans/` - Design documents

## Development

Open `index.html` directly in a browser - no build step needed.

## Implementation Notes

- **Game Engine:** Claude builds physics, collision, AI, game loop
- **UI/Visuals:** Gemini handles menus, effects, character design
- Pure vanilla JS, no frameworks
- 60fps target, Canvas API rendering

## Controls

| Player | Move | Weapons | Specials |
|--------|------|---------|----------|
| P1 | WASD | 1/2/3 | Q/E |
| P2 | Arrows | Numpad 7/8/9 | ,/. |

## Characters

- **Benioff:** Speed-focused, Cloud Bolt + Ohana Shield
- **McDermott:** Power-focused, Workflow Blast + Automation Drone

## Arenas

1. Cloud City (purple/pink)
2. Data Center (blue/cyan)
3. Boardroom (gold/white)
