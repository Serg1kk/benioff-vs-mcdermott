# Benioff vs McDermott: Death Match

A neon cyberpunk top-down arena shooter featuring tech CEO battles.

![Game Preview](preview.png)

## Play

Open `index.html` in any modern browser.

## Game Modes

- **Single Player** - Battle against AI (Easy/Normal/Hard)
- **Local Multiplayer** - 2 players on one keyboard

## Characters

| | Benioff | McDermott |
|---|---------|-----------|
| Style | Speed | Power |
| Special 1 | Cloud Bolt | Workflow Blast |
| Special 2 | Ohana Shield | Automation Drone |

## Controls

### Player 1
- **Move:** WASD
- **Weapons:** 1 (Blaster), 2 (Grenade), 3 (Dash)
- **Specials:** Q, E

### Player 2
- **Move:** Arrow Keys
- **Weapons:** Numpad 7/8/9
- **Specials:** , (comma), . (period)

## Arenas

1. **Cloud City** - Purple/pink neon, floating platforms
2. **Data Center** - Blue/cyan neon, server racks
3. **Boardroom** - Gold/white neon, corporate battlefield

## Tech

- Pure HTML5 Canvas + JavaScript
- No dependencies, no build step
- 60fps gameplay
